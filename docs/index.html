项目介绍

NameProject 是一款基于 Java 开发的随机点名器，旨在简化课堂管理流程。

  • 支持导入学生名单
  • 随机点名功能
  • 导出点名记录
  • 自定义点名顺序
  • 随机事件增加课堂氛围
  • 显示欢迎语缓解紧张气氛

无论是教师还是学生，都能从中获得便捷高效的体验。

功能

  • 支持导入学生名单
  • 随机点名功能
  • 导出点名记录
  • 自定义点名顺序
  • 随机事件增加课堂氛围
  • 显示欢迎语缓解紧张气氛

使用方法

  1. 启动NameProject点名器应用程序
  2. 导入学生名单
  3. 点击“开始抽取”按钮
  4. 点击“停止抽取”按钮结束点名
  5. 查看点名结果并进行记录
  6. 在设置界面中开启或关闭随机事件和欢迎语功能
