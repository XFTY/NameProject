其实，1.0是不太让我满意的，当然这和开发周期短有关，随着后面寒假带来的时间，我开始着手优化项目，基于我的想法在开发了新一代点名器。
#### 如何让点名变得更加不确定
我设计了一个“滚动条”，让名字从左往右依次流过屏幕，在用户选择“停止抽取”时让它依靠惯性缓缓停下，并最终确定人选。  
同时，我尽可能地美化了UI，不至于像90年代的风格（虽然依旧很像...）
